Content Policy Blueprint

Social Media Posting Guidelines & Strategy Framework

Content Policy Prompt

Here's my Social Media Content policy:

Ideal Uses:

1. Curiosity Post
   *Opens the loop, attracts attention, primes interest*

2. Value Post (No CTA)
   *Delivers goodwill and authority without asking*

3. Story Post
   *Humanizes you and lowers resistance*

4. Value Post (With CTA)
   *Now the CTA feels earned, not intrusive*

5. Question Post
   *Invites participation and resets the algorithm*

6. Hot Take Post
   *Creates momentum and discussion*

7. Win Post
   *Closes the loop with social proof*

| Stage            | What the Audience Feels                |
| ---------------- | -------------------------------------- |
| Curiosity        | “Interesting… tell me more”            |
| Value (No CTA)   | “This is useful”                       |
| Story            | “I relate to this person”              |
| Value (With CTA) | “I trust this enough to act”           |
| Question         | “I want to join the conversation”      |
| Hot Take         | “I have an opinion on this”            |
| Win              | “They clearly know what they’re doing” |

## Posting Frequency Example (Facebook Profile/Page)

### 5 Posts / Week (Recommended)

| Day        | Post Type                            |
| ---------- | -------------------------------------|

FOR NON-ADMIN PUBLIC GROUPS:

What to Post (and What NOT to Post):

✅ Allowed & Encouraged
Value posts (tactical, specific, helpful)
Question posts (genuine curiosity)
Light story posts (learning-focused)
Replies on others’ posts (very important)
⚠️ Hot takes only if clearly value-driven and aligned with group norms

❌ Avoid Completely
Sales posts
Win posts about yourself
Lead magnets
External links (unless explicitly allowed)
“DM me” CTAs

1, First Value Post:
Goal: Test acceptance
Post type: Value
Structure:
Start with group context (“Seeing many questions about X…”)
Share 3–5 actionable tips
End without CTA
❌ No links
❌ No selling
❌ No self-promotion

2, Second Value Post:
Goal: Reinforce expertise
Go deeper than first value post
Solve a problem raised in comments
Still no CTA

3, Question Post:
Goal: Invite conversation
Post type: Question
Examples:
“What’s the biggest challenge you’re facing with X?”
“Curious how others here handle Y.”

This signals humility and community focus.

Groups reward contribution. Profiles reward credibility. Don’t confuse the two.
