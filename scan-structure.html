Gumroad Product Setup Rules

Set up your Gumroad product - Key rules:
Title: must contain the RESULT, e.g. "28 Reels Hooks That Boost Watch Time This Week"
Description: 3–5 bullets explaining what the guide enables, not what's inside.
Tags: Use 5–10 tags that creators actually search: "Instagram hooks", "content ideas", "short-form content", "creator tools", "social media growth"
Price: Set min price = $3 (Pay-what-you-want with $3 minimum gives upside.)
